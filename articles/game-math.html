Game Mathematics: The Language of Motion

Game Dev üìÖ Updated: December 2024 ‚è±Ô∏è 20 min read

Mathematics is the foundation of game development. It allows us to describe positions, rotations, and physical interactions in a virtual world.

Vectors

Vectors are used to represent positions, directions, and velocities. A 3D vector has three components: X, Y, and Z.

C# 
// Vector addition
Vector3 a = new Vector3(1, 2, 3);
Vector3 b = new Vector3(4, 5, 6);
Vector3 c = a + b; // (5, 7, 9)

// Dot product (used for angles and projection)
float dot = Vector3.Dot(a, b);

// Cross product (used for finding normals)
Vector3 cross = Vector3.Cross(a, b);

Matrices & Quaternions

Matrices are used for transformations (translation, rotation, scaling). Quaternions are used specifically for rotations to avoid "gimbal lock".

Trigonometry

Sine, cosine, and tangent are essential for calculating angles, circular motion, and wave behaviors.
