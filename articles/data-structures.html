Data Structures

Computer Science üìÖ Updated: November 2024 ‚è±Ô∏è 25 min read

Data structures are specialized formats for organizing, processing, retrieving, and storing data. Choosing the right data structure can significantly improve the efficiency of an algorithm.

Arrays

An array is a collection of items stored at contiguous memory locations. It is the simplest data structure where each data element can be accessed directly by only using its index number.

Python 
# Array implementation in Python (List)
arr = [10, 20, 30, 40, 50]

# Accessing elements - O(1)
print(arr[2])  # Output: 30

# Searching - O(n)
if 40 in arr:
    print("Found")

Linked Lists

A linked list is a linear data structure where elements are not stored at contiguous memory locations. The elements in a linked list are linked using pointers.

Python 
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None
    
    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        last = self.head
        while last.next:
            last = last.next
        last.next = new_node

Stacks & Queues

Stacks and Queues are linear data structures with specific rules for adding and removing elements.

Stack (LIFO)

Last In, First Out. Think of a stack of plates.

  • Push: Add element to top
  • Pop: Remove element from top

Queue (FIFO)

First In, First Out. Think of a line at a store.

  • Enqueue: Add element to rear
  • Dequeue: Remove element from front

Trees

A tree is a hierarchical data structure consisting of nodes connected by edges.

Binary Search Tree (BST)

A binary tree where the left child is smaller than the parent, and the right child is greater.

Python 
class TreeNode:
    def __init__(self, val=0):
        self.val = val
        self.left = None
        self.right = None

def insert(root, val):
    if not root:
        return TreeNode(val)
    if val < root.val:
        root.left = insert(root.left, val)
    else:
        root.right = insert(root.right, val)
    return root

Hash Tables

Hash tables store data in key-value pairs. They use a hash function to compute an index into an array of buckets or slots.

Time Complexity: Average O(1) for insertion, deletion, and search.

Graphs

A graph is a non-linear data structure consisting of nodes (vertices) and edges. Graphs are used to represent networks.

  • Directed Graph: Edges have direction
  • Undirected Graph: Edges have no direction
  • Weighted Graph: Edges have weights/costs
