Solidity: Smart Contract Language

Blockchain üìÖ Updated: December 2024 ‚è±Ô∏è 20 min read

Solidity is an object-oriented, high-level language for implementing smart contracts. It is influenced by C++, Python, and JavaScript.

Introduction

Solidity is statically typed, supports inheritance, libraries, and complex user-defined types. It is designed to target the Ethereum Virtual Machine (EVM).

Basic Syntax

A Solidity file starts with a pragma directive to specify the compiler version.

Solidity 
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract SimpleStorage {
    uint256 storedData;

    function set(uint256 x) public {
        storedData = x;
    }

    function get() public view returns (uint256) {
        return storedData;
    }
}

Data Types

Solidity provides several elementary types: booleans, integers, addresses, and byte arrays. It also supports complex types like mappings and structs.

Solidity 
struct User {
    string name;
    address wallet;
}

mapping(address => User) public users;
