Shader Programming: Painting with Code

Shaders are small programs that run on the GPU. they are responsible for calculating the color of pixels and the position of vertices in 3D space.

Introduction

Modern graphics APIs use shaders for almost everything. The most common shader languages are HLSL (DirectX), GLSL (OpenGL), and Metal (Apple).

Vertex & Fragment Shaders

Vertex shaders handle the geometry, while fragment (or pixel) shaders handle the final color of each pixel.

GLSL 
// A simple fragment shader that outputs red
#version 330 core
out vec4 FragColor;

void main()
{
    FragColor = vec4(1.0f, 0.0f, 0.0f, 1.0f);
}

Lighting Models

Shaders are used to implement lighting models like Lambert, Phong, and PBR (Physically Based Rendering).
