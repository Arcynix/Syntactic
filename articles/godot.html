Godot Engine: The Open Source Alternative

Game Dev üìÖ Updated: December 2024 ‚è±Ô∏è 15 min read

Godot is a free, all-in-one, cross-platform game engine that makes it easy for you to create 2D and 3D games.

Introduction

Godot provides a huge set of common tools, so you can just focus on making your game without reinventing the wheel. It's completely free and open-source under the MIT license.

GDScript

GDScript is a high-level, dynamically typed programming language used to create content in Godot. It uses a syntax similar to Python.

GDScript 
extends KinematicBody2D

var speed = 200
var velocity = Vector2()

func _physics_process(delta):
    velocity = Vector2()
    if Input.is_action_pressed("ui_right"):
        velocity.x += 1
    if Input.is_action_pressed("ui_left"):
        velocity.x -= 1
    
    velocity = velocity.normalized() * speed
    move_and_slide(velocity)

Nodes & Scenes

Everything in Godot is a Node. Nodes are organized into Scenes, and Scenes can be instanced into other Scenes, creating a powerful hierarchical structure.
