TypeScript is a strongly typed programming language that builds on JavaScript, giving you better tooling at any scale.

Introduction

TypeScript adds additional syntax to JavaScript to support a tighter integration with your editor. Catch errors early in your editor.

Basic Types

TypeScript provides several basic types to help you define your data.

TypeScript 
let isDone: boolean = false;
let decimal: number = 6;
let color: string = "blue";
let list: number[] = [1, 2, 3];
let x: [string, number] = ["hello", 10]; // Tuple

Interfaces

One of TypeScript's core principles is that type checking focuses on the shape that values have.

TypeScript 
interface User {
    name: string;
    id: number;
}

const user: User = {
    name: "Hayes",
    id: 0,
};

Generics

Generics allow you to create reusable components that can work over a variety of types rather than a single one.

TypeScript 
function identity(arg: Type): Type {
    return arg;
}

let output = identity("myString");
