Swift: Fast, Safe, Expressive

Swift is a powerful and intuitive programming language for iOS, iPadOS, macOS, tvOS, and watchOS.

Introduction

Swift is a result of the latest research on programming languages, combined with decades of experience building Apple platforms.

Basics

Swift is easy to read and write, with a syntax that's clean and concise.

Swift 
let message = "Hello, Swift!"
var count = 10
count += 1

print("\(message) Count is \(count)")

Optionals

Optionals are a powerful feature in Swift that handle the absence of a value.

Swift 
var optionalName: String? = "John Doe"
optionalName = nil

if let name = optionalName {
    print("Hello, \(name)")
} else {
    print("Hello, Guest")
}
