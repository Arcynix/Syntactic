Responsive Web Design

Web üìÖ Updated: December 2024 ‚è±Ô∏è 15 min read

Responsive Web Design (RWD) is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes.

Introduction

RWD uses "media queries" to figure out what resolution of device it's being served on. This allows the website to adapt its layout to the screen size.

Media Queries

Media queries allow you to apply CSS styles depending on a device's general type or specific characteristics and parameters.

CSS 
/* Desktop first */
.container {
    width: 1200px;
}

/* Tablet */
@media (max-width: 1024px) {
    .container {
        width: 100%;
        padding: 0 20px;
    }
}

/* Mobile */
@media (max-width: 768px) {
    .nav-menu {
        display: none;
    }
}

Viewport Meta Tag

To ensure your site is responsive, you must include the viewport meta tag in your HTML head.

HTML 
<meta name="viewport" content="width=device-width, initial-scale=1.0">
