Blockchain Basics: Decentralized Trust

Blockchain ðŸ“… Updated: December 2024 â±ï¸ 15 min read

A blockchain is a distributed ledger that records transactions across many computers so that the record cannot be altered retroactively.

Introduction

Blockchain technology allows for decentralized trust, meaning that no single entity has control over the entire network. This is achieved through consensus mechanisms and cryptography.

How it Works

Transactions are grouped into "blocks", and each block contains a cryptographic hash of the previous block, creating a "chain".

JSON 
{
    "index": 1,
    "timestamp": "2024-12-20T14:45:30Z",
    "transactions": [
        {
            "sender": "Alice",
            "recipient": "Bob",
            "amount": 10
        }
    ],
    "previous_hash": "0000000000000000",
    "hash": "a1b2c3d4e5f6g7h8"
}

Consensus Mechanisms

Consensus mechanisms like Proof of Work (PoW) and Proof of Stake (PoS) ensure that all nodes in the network agree on the state of the ledger.
