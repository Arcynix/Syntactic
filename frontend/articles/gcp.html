Google Cloud Platform (GCP), offered by Google, is a suite of cloud computing services that runs on the same infrastructure that Google uses internally for its end-user products.

Introduction

GCP provides a series of modular cloud services including computing, data storage, data analytics and machine learning, alongside a set of management tools.

Core Services

  • Compute Engine: Virtual machines running in Google's data centers.
  • Cloud Storage: Unified object storage for all your data.
  • BigQuery: Serverless, highly scalable, and cost-effective multi-cloud data warehouse.
  • App Engine: Platform for building scalable web applications and mobile backends.

GKE (Google Kubernetes Engine)

GKE is a managed, production-ready environment for deploying containerized applications. It is often considered the best managed Kubernetes service because Kubernetes was originally developed at Google.
