Redis: In-Memory Data Store

Redis is an open source, in-memory data structure store, used as a database, cache, and message broker.

Introduction

Redis provides data structures such as strings, hashes, lists, sets, sorted sets with range queries, bitmaps, hyperloglogs, geospatial indexes, and streams.

Caching Patterns

Redis is most commonly used for caching to reduce database load and improve application performance.

Redis 
# Set a key with expiration
SET user:1001 "John Doe" EX 3600

# Get a key
GET user:1001

# Increment a counter
INCR page_views:home

Pub/Sub

Redis also supports the Publish/Subscribe messaging paradigm.

Redis 
# Subscriber
SUBSCRIBE news_channel

# Publisher
PUBLISH news_channel "Hello World!"
