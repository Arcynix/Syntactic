Unity is a cross-platform game engine developed by Unity Technologies. It is used to create half of the world's games.

Introduction

Unity's primary programming language is C#. It features a powerful editor that allows for real-time iteration and a vast asset store.

Scripting in C#

Scripts in Unity are components that you attach to GameObjects to define their behavior.

C# 
using UnityEngine;

public class PlayerController : MonoBehaviour
{
    public float speed = 5.0f;

    void Update()
    {
        float moveHorizontal = Input.GetAxis("Horizontal");
        float moveVertical = Input.GetAxis("Vertical");

        Vector3 movement = new Vector3(moveHorizontal, 0.0f, moveVertical);
        transform.Translate(movement * speed * Time.deltaTime);
    }
}

Physics System

Unity has a built-in physics engine (NVIDIA PhysX) that handles collisions, gravity, and other physical forces.

C# 
void OnCollisionEnter(Collision collision)
