Rust is a multi-paradigm, general-purpose programming language designed for performance and safety, especially safe concurrency.

Introduction

Rust is syntactically similar to C++, but it can guarantee memory safety by using a borrow checker to validate references.

Ownership and Borrowing

Ownership is Rust's most unique feature, which enables Rust to make memory safety guarantees without needing a garbage collector.

Rust 
fn main() {
    let s1 = String::from("hello");
    let s2 = s1; // s1 is moved to s2

    // println!("{}, world!", s1); // This would cause a compile error
    println!("{}, world!", s2);
}

Concurrency

Rust's ownership and type systems are a powerful tool for helping to manage memory and concurrency problems.

Rust 
use std::thread;

fn main() {
    let handle = thread::spawn(|| {
        println!("Hello from a thread!");
    });

    handle.join().unwrap();
}
