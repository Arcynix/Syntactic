Machine Learning: Learning from Data

AI/ML üìÖ Updated: December 2024 ‚è±Ô∏è 20 min read

Machine learning (ML) is a field of inquiry devoted to understanding and building methods that 'learn', that is, methods that leverage data to improve performance on some set of tasks.

Introduction

ML is a subset of artificial intelligence. It involves the use of algorithms and statistical models to perform a specific task without using explicit instructions, relying on patterns and inference instead.

Types of Learning

  • Supervised Learning: The algorithm is trained on a labeled dataset (e.g., classification, regression).
  • Unsupervised Learning: The algorithm is trained on an unlabeled dataset and must find patterns on its own (e.g., clustering).
  • Reinforcement Learning: The algorithm learns by interacting with an environment and receiving rewards or penalties.

ML in Python

Python is the most popular language for machine learning, thanks to libraries like Scikit-Learn, TensorFlow, and PyTorch.

Python 
from sklearn.linear_model import LinearRegression
import numpy as np

# Sample data
X = np.array([[1], [2], [3], [4]])
y = np.array([2, 4, 6, 8])

# Create and train the model
model = LinearRegression()
model.fit(X, y)

# Make a prediction
prediction = model.predict([[5]])
print(prediction) # [10.]
