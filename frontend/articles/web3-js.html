Web3.js & Ethers.js: Connecting to the Blockchain

Blockchain üìÖ Updated: December 2024 ‚è±Ô∏è 15 min read

Web3.js and Ethers.js are JavaScript libraries that allow you to interact with a local or remote Ethereum node using HTTP, IPC, or WebSocket.

Introduction

These libraries are essential for building decentralized applications (DApps). They provide a way to send transactions, call smart contract functions, and listen for events on the blockchain.

Using Ethers.js

Ethers.js is a popular, lightweight alternative to Web3.js. It is known for its simple API and strong focus on security.

JavaScript 
import { ethers } from "ethers";

// Connect to the provider (e.g., MetaMask)
const provider = new ethers.providers.Web3Provider(window.ethereum);

// Get the signer (the user's wallet)
const signer = provider.getSigner();

// Interact with a contract
const contract = new ethers.Contract(address, abi, signer);
const balance = await contract.balanceOf(userAddress);
console.log(ethers.utils.formatEther(balance));

Using Web3.js

Web3.js is the original library for Ethereum development. It is feature-rich and widely used in legacy projects.

JavaScript 
import Web3 from "web3";

const web3 = new Web3(window.ethereum);
const accounts = await web3.eth.requestAccounts();
const balance = await web3.eth.getBalance(accounts[0]);
console.log(web3.utils.fromWei(balance, "ether"));
