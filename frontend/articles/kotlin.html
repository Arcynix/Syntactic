Kotlin: Modern, Concise, Safe

Language ðŸ“… Updated: December 2024 â±ï¸ 20 min read

Kotlin is a modern, cross-platform, statically typed, general-purpose programming language with type inference.

Introduction

Kotlin is designed to interoperate fully with Java, and the JVM version of Kotlin's standard library depends on the Java Class Library.

Basics

Kotlin is concise, reducing the amount of boilerplate code you need to write.

Kotlin 
fun main() {
    val name = "Kotlin"
    println("Hello, $name!")
}

Null Safety

Kotlin's type system is aimed at eliminating the danger of null references from code, also known as the "billion-dollar mistake".

Kotlin 
var a: String = "abc" // Regular initialization means non-null by default
// a = null // compilation error

var b: String? = "abc" // can be set to null
b = null // ok
println(b?.length)
