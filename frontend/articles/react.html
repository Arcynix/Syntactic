React is a JavaScript library for building user interfaces, maintained by Meta (formerly Facebook). It allows developers to create large web applications that can change data, without reloading the page.

Core Concepts

React is built around the concept of components - reusable pieces of UI that manage their own state.

Hooks

Hooks were introduced in React 16.8 to let you use state and other React features without writing a class.
