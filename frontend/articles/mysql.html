MySQL: The World's Most Popular Open Source Database

Database ðŸ“… Updated: December 2024 â±ï¸ 15 min read

MySQL is an open-source relational database management system (RDBMS). Its name is a combination of "My", the name of co-founder Michael Widenius's daughter, and "SQL", the abbreviation for Structured Query Language.

Introduction

MySQL is a central component of the LAMP open-source web application software stack (and others such as WAMP, MAMP, and SAMP).

Basic Commands

MySQL uses standard SQL syntax with some specific extensions.

SQL 
-- Create a database
CREATE DATABASE my_app;

-- Use the database
USE my_app;

-- Create a table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Indexing

Indexes are used to find rows with specific column values quickly.

SQL 
CREATE INDEX idx_username ON users(username);
