SQL: Structured Query Language

Database üìÖ Updated: December 2024 ‚è±Ô∏è 15 min read

SQL is the standard language for dealing with Relational Databases. It is used to insert, search, update, and delete database records.

Introduction

SQL is used to communicate with a database. According to ANSI (American National Standards Institute), it is the standard language for relational database management systems.

Basic Queries

The most common SQL commands are SELECT, INSERT, UPDATE, DELETE, and CREATE.

SQL 
-- Select all columns from Users table
SELECT * FROM Users;

-- Filter results
SELECT name, email 
FROM Users 
WHERE age > 18;

-- Insert new record
INSERT INTO Users (name, email, age)
VALUES ('John Doe', 'john@example.com', 25);

Joins

A JOIN clause is used to combine rows from two or more tables, based on a related column between them.

SQL 
SELECT Orders.OrderID, Customers.CustomerName
FROM Orders
INNER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;
